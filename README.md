Tech Stack

Node.js - runtime environment for the backend.
TypeScript - static typing for safer code and better tooling.
Express v5 - HTTP server and routing for the API.
Prisma - ORM for database access, migrations, and models.
PostgreSQL + pg - relational database and its driver.
Zod - request validation for incoming data.
Pino + pino-http - structured logging and HTTP request logs.
CORS + cookie-parser - cross‑origin support and cookie parsing.
bcrypt - secure password hashing.
jsonwebtoken - JWT authentication.
ts-node-dev - fast dev server with auto‑reload.
